\section{Conclusion}
This project delivers a complete and practical Futoshiki-solving system that combines multiple AI approaches with an interactive GUI. Through implementation and testing, the team verified that different paradigms (search, logic inference, and SAT) can all be applied to the same constraint problem with different trade-offs in runtime, memory, and node expansions.

From an engineering perspective, the project codebase is organized for reproducibility and extension, including standardized input/output handling, solver abstractions, and automated testing. The implementation environment is lightweight and aligned with the project requirements, using \texttt{pygame} for GUI interaction, \texttt{python-sat} for SAT solving, and \texttt{pytest} for test execution and validation.

Overall, the project objectives were achieved: modeling constraints correctly, implementing representative AI methods, evaluating solver behavior across benchmark inputs, and presenting results through both command-line tests and GUI-based visualization.
